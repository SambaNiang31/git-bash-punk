How to use vim:


:w pour sauvegarder
:wq pour sauvegarder et quitter ( ou :x)
:q! pour quitter sans sauvegarder
dd pour supprimer une ligne
u pour annuler (undo)
	P pour coller a la ligne supérieure
	p pour coller a la ligne inférieure


How tu use pwd:

--> clear to clear the console
--> ls to see in which folder you are
--> ls -a to see hiden folder
--> mv to move and rename
--> cat 'ooo' pour afficher le fichier 'ooo'
--> rm -rd 'name' to delete 'name'
